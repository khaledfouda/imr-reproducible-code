# Data Directory

This directory contains input data for the two applications in the paper.

## `movielens/raw/`

Raw MovieLens 1M data files, included in this repository. Derived from the
[MovieLens 1M dataset](https://grouplens.org/datasets/movielens/1m/) (Harper &
Konstan, 2015).

The following files are included in this directory:

| File | Contents |
|------|----------|
| `Movie_Y.Rdata` | training user–movie ratings |
| `Movie_X.Rdata` | User covariates: gender and age group indicators |
| `Movie_Q.Rdata` | test set ratings |
| `movies_Z.dat` | Movie covariates: genre labels |

The following two files are generated by running the script `2_1_prepare_python_data.R`: 
| `Movie_Y_Glocalk.dat` | training user–movie ratings for Glocal-K |
| `Movie_test_Glocalk.dat` | test set ratings for Glocal-K |


## `Bixi/train_test_splits`

The generated train/test splits for the Bixi dataset are stored in this directory. Run the script `1_generate_train_test_splits.R` in `code/bixi/` to generate these splits.
