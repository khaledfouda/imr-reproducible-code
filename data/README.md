# Data Directory

This directory contains input data for the two applications in the paper.

## `movielens/raw/`

The preprocessed MovieLens 1M data files used in the paper. These files are taken
as-is from the supplementary material of Ma et al. (2025), which preprocesses the
original [MovieLens 1M dataset](https://grouplens.org/datasets/movielens/1m/) into the response and covariate matrices below. We did not
modify them, so no additional preprocessing code is required.

> Ma, S., Niu, P. Y., Zhang, Y., & Zhu, Y. (2025). Statistical Inference for Noisy
> Matrix Completion Incorporating Auxiliary Information. *Journal of the American
> Statistical Association*, 120(549), 343–355.
> https://doi.org/10.1080/01621459.2024.2335591

| File | Contents |
|------|----------|
| `Movie_Y.Rdata` | training user–movie ratings |
| `Movie_X.Rdata` | User covariates: gender and age group indicators |
| `Movie_Q.Rdata` | test set ratings |
| `movies_Z.dat` | Movie covariates: genre labels |

The following two files are generated by running the script `2_1_prepare_python_data.R`:

| File | Contents |
|------|----------|
| `Movie_Y_Glocalk.dat` | training user–movie ratings for Glocal-K |
| `Movie_test_Glocalk.dat` | test set ratings for Glocal-K |


## `Bixi/train_test_splits`

The generated train/test splits for the Bixi dataset are stored in this directory. Run the script `1_generate_train_test_splits.R` in `code/Bixi/` to generate these splits.

The underlying Bixi data (trip counts and the spatial and temporal covariates) is
provided by the `BKTR` package via `BixiData$new()`; see the BKTR documentation for
the variable definitions: https://cran.r-project.org/package=BKTR
